Magnolia community webapp pre-bundled with the DAM module and Magnolia Templating Essentials.

This package contains:

   Magnolia Core 5.6.7
   magnolia-empty-webapp 5.6.8
   magnolia-empty-webapp 5.6.8
   Magnolia Internationalization Framework 5.6.7
   Magnolia JAAS 5.6.7
   Google Sitemap Module 2.5
   magnolia-module-mail 5.5
   magnolia-module-rssaggregator 2.6
   Magnolia Rendering 5.6.7
   Magnolia Templating 5.6.7
   Magnolia Templating for JSP 5.6.7
